# 📋 Lista de Tarefas

Uma aplicação simples de **Lista de Tarefas (To-Do List)** desenvolvida com **HTML, CSS e JavaScript**, permitindo adicionar, concluir e remover tarefas de forma dinâmica.

## 🚀 Demonstração

O projeto permite:

* ➕ Adicionar novas tarefas
* ✅ Marcar tarefas como concluídas
* 🗑️ Excluir tarefas
* ⌨️ Adicionar tarefas pressionando **Enter**
* 📱 Interface simples e responsiva

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Material Design Icons

## ⚙️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/Lista-de-Tarefas.git
```

2. Entre na pasta do projeto:

```bash
cd Lista-de-Tarefas
```

3. Abra o arquivo `index.html` no navegador.

> Não é necessário instalar dependências ou executar um servidor.

---

## 💻 Funcionalidades

* Adição dinâmica de tarefas.
* Marcação visual de tarefas concluídas.
* Organização das tarefas concluídas ao final da lista.
* Remoção individual de tarefas.
* Suporte à tecla **Enter** para adicionar novas tarefas.
